An offline reader for Boeing maintenance manuals — AMM, AIPC, WDM and FIM — on Windows or Android, no connection needed. First fleet: the Air Algérie B737-89.

Download for Windows Installer · — · — Download for Android APK · — · —

These builds are not code-signed yet. On Windows, SmartScreen will warn “unknown publisher” — click More info → Run anyway. The Android APK is a side-load build; enable Install unknown apps for your browser. Verify the sha256 listed on the release page.

Fully offline

The documentation package lives on the device. No server, no connection needed at read time — built for the hangar floor.

Built for the print

AMM, AIPC, WDM and FIM with the task codes, figures, parts and wiring you already know — navigated the way the manuals are organised.

Cross-reference aware

Tasks, parts, wires and effectivity resolve to each other and read in their reference colours, so a citation is one tap from its target.
